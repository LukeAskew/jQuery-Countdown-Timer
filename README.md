jQuery-Countdown-Timer
======================

Super awesome timer that counts down to certain time and date.

### Usage ###

  <div id="timer">
		<div class="hours"></div>
		<div class="minutes"></div>
		<div class="seconds"></div>
	</div>

	<script>
		// Time must be UTC/GMT  
		$('#timer').countdown('June 06, 2012 15:00:00');
	</script>
